Algunos de mis trabajos

Java Script

JavaScript es un lenguaje clave para el desarrollo web, usado para crear interactividad y dinamismo en páginas. Funciona en cliente y servidor con herramientas como Node.js, es versátil, abierto y compatible con navegadores actuales.

Formatos CSS

CSS da estilo a las páginas web, controlando colores, fuentes, márgenes y diseño. Es clave para crear sitios atractivos y adaptables a cualquier dispositivo.

Flexbox and Grid

Flexbox y CSS Grid son herramientas de diseño en CSS. Flexbox organiza elementos en una sola dirección (fila o columna), mientras que CSS Grid permite crear diseños bidimensionales con filas y columnas. Ambos facilitan estructuras adaptables para distintos tamaños de pantalla.
